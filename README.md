# letter_yu
Лабораторная работа: 3D Визуализация и преобразования
Описание проекта
Веб-приложение для визуализации трехмерной модели буквы "Ю" с реализацией трехмерных преобразований и ортографических проекций. Проект включает три лабораторные работы:

Лаба 6a - Построение и визуализация трехмерных объектов (каркасная модель)

Лаба 6b - Реализация трехмерных преобразований (масштабирование, перенос, вращение)

Лаба 6c - Построение ортографических проекций (Oxy, Oxz, Oyz)

Технологии
HTML5, CSS3, JavaScript - базовые технологии веб-разработки

Three.js r128 - библиотека для 3D графики в браузере

Docker - контейнеризация приложения

Nginx - веб-сервер для раздачи статических файлов

Функциональность
1. 3D Модель буквы "Ю"
Построена из трех частей: вертикальная линия, горизонтальный соединитель, круг

Реализована как THREE.Group с тремя дочерними мешами

Поддержка двух режимов отображения: каркасный и сплошной

2. Трехмерные преобразования (Лаба 6b)
Масштабирование - равномерное изменение размера

Перенос - перемещение по осям X, Y, Z

Вращение - вокруг основных осей (X, Y, Z) и произвольной оси

Матрица преобразования 4×4 отображается в реальном времени

3. Ортографические проекции (Лаба 6c)
Проекция XY - вид сверху (зеленый)

Проекция XZ - вид спереди (фиолетовый)

Проекция YZ - вид сбоку (желтый)

Переключение между видами через вкладки

4. Графический интерфейс
3D сцена с системой координат

Панель управления с ползунками и кнопками

Информационная панель с данными о модели

Отображение матрицы преобразования

Алгоритмы реализации
1. Построение 3D модели
Использование THREE.BoxGeometry для линий

Использование THREE.TorusGeometry для круга

Объединение в THREE.Group для единого управления

2. Матричные преобразования
Реализация умножения матриц 4×4

Генерация матриц масштабирования, переноса, вращения

Композиция преобразований: T × R × S

3. Ортографические проекции
Использование THREE.OrthographicCamera

Настройка камер для каждой координатной плоскости

Автоматическое переключение видов

Установка и запуск
Способ 1: Локальный запуск
Скопируйте файлы в папку проекта

Откройте index.html в браузере

Убедитесь что файл three.min.js находится в папке js/

Способ 2: Запуск через Docker
bash
# Сборка и запуск контейнера
docker-compose up --build -d

# Проверка статуса
docker ps

# Остановка контейнера
docker-compose down
Приложение будет доступно по адресу: http://localhost:8081

Управление приложением
Основные элементы управления:
Вращение сцены - зажатая левая кнопка мыши + движение

Приближение/отдаление - колесико мыши

Перемещение сцены - зажатая правая кнопка мыши + движение

Панель управления:
Масштаб - ползунок для изменения размера модели

Перенос - ползунки для перемещения по осям X, Y, Z

Вращение - угол вращения и выбор оси

Цвета - настройка цветов объекта и фона

Режимы - переключение каркасного/сплошного режима

Вкладки проекций:
3D Объект - полная трехмерная модель

Проекция XY - вид сверху

Проекция XZ - вид спереди

Проекция YZ - вид сбоку

Требования к системе
Минимальные требования:
Браузер с поддержкой WebGL (Chrome 60+, Firefox 55+, Edge 79+)

1 ГБ оперативной памяти

Видеокарта с поддержкой OpenGL ES 2.0

Для Docker:
Docker Desktop 2.0+

2 ГБ свободного места на диске

Включенная виртуализация в BIOS

Разработка и модификация
Структура кода:
GraphicsLab.init() - инициализация приложения

createSimpleLetter() - создание 3D модели

applyCurrentTransformations() - применение преобразований

updateMatrixDisplay() - обновление матрицы

switchScene() - переключение между проекциями

Добавление новых функций:
Для новой трансформации - добавить в updateTransformationMatrix()

Для новой проекции - добавить в switchScene()

Для нового элемента управления - добавить в setupControls()
